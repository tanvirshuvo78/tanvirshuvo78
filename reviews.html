Fifa 22 (2021)

Electronic Arts

Asad Sunny

This game is the best fifa instalment since fifa 17, people are complaining about things like ‘scripting’ but that’s because they don’t win every game, also it’s much harder to score this year with goalkeepers like this, but that’s the thing, it shouldn’t be easy, if it was so easy to score in real life there would be so much less emotion in the crowd when a goal is scored. Another good thing is that EA have made slower players with 70-75 much more ‘usable’ because of the way the game feels. A bad thing though, is that EA still has fifa points, so it can still be considered pay to win. But honestly, if you want to get as much enjoyment out of ultimate team, at least for me the best option is to go road to glory style. Career mode has been massively improved, with a complete refresh of player career and create a club being a returning game mode for the first time in YEARS. Normal manager career hasn’t been changed, but it’s still a very enjoyable gamemode, so no complaints from me. I haven’t tried pro clubs, seasons or Volta yet, but it looks like EA have made significant changes to all three game modes. Thank you for reading my opinion on this really great game, and I hope that this can influence you.
